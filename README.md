# 🌍 Multi-Agent Travel Planner (MAS)

## 📌 Description
This project implements a Multi-Agent System (MAS) using CrewAI and Ollama to automate travel planning.

The system uses multiple AI agents to:
- Find tourist places
- Suggest hotels
- Calculate budget
- Generate a final itinerary

## 🧠 Technologies
- CrewAI
- Ollama (Local LLM)
- Python
- JSON (local dataset)

## 🤖 Agents
1. Research Agent – finds places
2. Hotel Agent – finds hotels
3. Budget Agent – calculates cost
4. Report Agent – generates travel plan

## 🧰 Tools
- travel_tool → reads places
- hotel_tool → reads hotels
- cost_tool → calculates cost
- report_tool → generates report
